# Стек мониторинга и логирования (Observability)

Комплексная автоматизация развертывания системы мониторинга (Zabbix) и централизованного сбора логов (ELK) с упором на безопасность.

## Ключевые компоненты
* **Логирование (ELK):** Централизованное хранилище на базе Elasticsearch и визуализация в Kibana и Grafana.
* **Мониторинг (Zabbix):** Автоматизированное развертывание сервера и агентов Zabbix 7.0.
* **Единая визуализация:** Дашборды в **Grafana**, объединяющие метрики из Zabbix и логи из Elastic.

## Безопасность и надежность
* **Full-Stack TLS:** Автоматизированная PKI-инфраструктура (свой CA) для шифрования трафика между всеми узлами.
* **Управление секретами:** Безопасное хранение паролей через **Ansible Vault** и внутренние Keystores (Elastic/Kibana).
* **Изоляция:** Запуск Filebeat от непривилегированного пользователя и кастомные политики **SELinux**.
* **Идемпотентность:** Использование маркерных файлов и проверок состояний для стабильного повторного запуска плейбуков.

## Технологический стек
* **Сбор логов:** Filebeat
* **БД:** MariaDB
* **Web:** Nginx (Reverse Proxy с SSL-терминацией)
* **ОС:** AlmaLinux 9
